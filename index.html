📖 使用指南

🎙️ 语音记录功能

  1. 点击左侧导航栏的「语音记录」按钮进入页面
  2. 点击页面中央的麦克风按钮开始录音
  3. 清晰说出日期、时间和具体事情
  4. 录音完成后，AI会自动整理成结构化日记
  5. 点击「保存到成长日记」按钮保存记录

📸 照片上传功能

  1. 点击左侧导航栏的「照片上传」按钮进入页面
  2. 点击或拖拽照片到上传区域
  3. AI会自动分析照片内容、生成标签和情绪识别
  4. 查看AI分析结果，确认无误后点击保存
  5. 照片会自动添加到成长日记中

💝 记忆胶囊功能

  1. 点击左侧导航栏的「记忆胶囊」按钮进入页面
  2. 选择「生成新胶囊」自动汇总近期记录
  3. 或选择「时间胶囊」设定未来开启时间
  4. AI会生成专属的成长报告和寄语
  5. 可下载保存记忆胶囊作为珍贵纪念

🤖 AI顾问功能

  1. 点击左侧导航栏的「AI顾问」按钮进入页面
  2. 在输入框中输入您的问题
  3. 或点击热门问题快速提问
  4. AI顾问会根据孩子情况提供个性化建议
  5. 可随时咨询训练方法、情绪管理等问题

🎵 背景音乐功能

  1. 点击右下角的音乐按钮播放/暂停背景音乐
  2. 背景音乐为温馨的卡农钢琴曲，营造舒适氛围
  3. 音乐音量默认为15%，不会打扰您的使用
  4. 可随时关闭音乐，不影响其他功能使用
